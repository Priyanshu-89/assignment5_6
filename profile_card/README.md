#  Profile Card React App

##  Assignment 1: Build a Simple Profile Card

###  Objective
Practice using **JSX**, **Functional Components**, **Props**, and **Styling** in React.

---

##  Features
- Displays multiple **profile cards** with:
  - Avatar (image)
  - Name
  - Role/Position
- Includes a **"Say Hello"** button that triggers an alert message:  
  `alert("Hello from " + name);`
- Simple and responsive layout using **inline CSS (Flexbox)**.

---

##  Technologies Used
- **React.js**
- **JavaScript (ES6)**
- **JSX**
- **Inline CSS**
